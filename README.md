# Judy Community Post Frontend

게시물 관리를 위한 프론트엔드 애플리케이션입니다.

## 구조

```
judy-community-frontend-post/
├── public/
├── src/
│   ├── components/
│   ├── pages/
│   ├── services/
│   ├── App.js
│   ├── index.js
│   ├── styles/
│   └── utils/
├── package.json
├── .env
└── README.md
```

## 기능

- 게시물 생성 페이지
- 게시물 목록 페이지
- 게시물 상세 페이지

## 설치 및 실행

### 사전 요구 사항

- [Node.js](https://nodejs.org/)

### 애플리케이션 설치 및 실행

```bash
npm install
npm start
```
